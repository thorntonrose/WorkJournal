Work Journal
Copyright 2001, Thornton Rose

Work Journal is a small application that is used to keep a journal of work
activity - date, start time, end time, etc. It is written in Java and has a 
Swing GUI.

Documentation:

   * Manual - docs/manual.html
   * Javadocs - docs/api/index.html
   * Version history - docs/history.txt

Dependencies:

   * JDK 1.3
   * JavaCogs 1.6 (lib/javacogs.jar)
   * HSQLDB 1.6 (lib/hsqldb.jar)
